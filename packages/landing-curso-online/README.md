# 🎓 Landing Page - Curso Online

Template profissional otimizado para venda de cursos online e infoprodutos.

## ✨ Características

- Design agressivo de conversão
- Urgência e escassez integradas
- Seção de depoimentos com resultados
- Módulos expansíveis
- Pricing com bônus destacados
- FAQ interativo
- 100% responsivo

## 🎯 Ideal Para

- Cursos online
- Infoprodutos
- Mentorias em grupo
- Workshops gravados
- Programas de transformação

## 📊 Elementos de Conversão

✅ Hero com vídeo de vendas  
✅ Social proof (número de alunos, rating)  
✅ Módulos detalhados  
✅ Depoimentos com resultados  
✅ Pricing com urgência  
✅ Garantia destacada  
✅ Múltiplos CTAs

## 🚀 Como Usar

```bash
cd landing-curso-online
npm install
npm run dev
```

## 🎨 Customizar

1. **Conteúdo:** Edite `content.js`
2. **Cores:** Ajuste `tailwind.config.js` (tema purple/indigo)
3. **Vídeo:** Adicione YouTube ID em `content.hero.videoId`

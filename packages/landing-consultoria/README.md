# 🎓 Landing Page - Consultoria / Coaching

Template profissional otimizado para venda de consultoria, coaching e programas de transformação pessoal/profissional.

## ✨ Características

- Design agressivo de conversão
- Copy com foco em transformação (não features)
- Seção de depoimentos com resultados específicos
- Stack de valor visual
- Garantia forte (reversão de risco)
- Urgência/Escassez estratégica
- FAQ interativo
- 100% responsivo mobile-first
- Integração fácil com CRM/Email

## 🎯 Ideal Para

- Consultores de negócios
- Coaches de vida/carreira
- Programas de transformação
- Mentoria 1-on-1 ou em grupo
- Treinamentos corporativos
- Cursos de desenvolvimento pessoal

## 🚀 Como Usar

### 1. Instalar dependências
```bash
npm install
```

### 2. Rodar desenvolvimento
```bash
npm run dev
```

Acesse: http://localhost:3000

### 3. Build para produção
```bash
npm run build
```

## 🎨 Customizar

### Editar Conteúdo
Abra `content.js` e modifique:
- Headline e subheadline
- Textos da dor e solução
- Benefícios
- Depoimentos (substitua placeholders)
- Preços e oferta
- Garantia e urgência
- FAQ

### Mudar Cores
Edit `tailwind.config.js`:
```javascript
colors: {
  primary: '#1e40af',    // Azul
  secondary: '#7c3aed',  // Roxo
  accent: '#f59e0b',     // Âmbar
}
```

### Adicionar Componentes
1. Crie novo arquivo em `src/components/`
2. Importe em `src/App.jsx`
3. Estilize com Tailwind

## 🔧 Troubleshooting

**Porta 3000 ocupada:**
```bash
npm run dev -- --port 3001
```

**Problema com Tailwind:**
```bash
rm -rf node_modules package-lock.json
npm install
npm run dev
```

## 🚀 Como Começar

### Para Usuários Finais (Consultores/Coaches)

1. Escolha o template que cabe seu serviço
2. Customize `content.js` com seus dados
3. Rode `npm install && npm run dev`
4. Preview local em http://localhost:3000
5. Deploy em Vercel/Netlify/GitHub Pages
6. Aponte seu domínio
7. Comece a rodar tráfego pago

### Para Agências/White Label

1. Clone o repositório
2. Customize templates para seus clientes
3. Adicione seu branding/logo
4. Deploy em subdomínio ou domínio cliente
5. Fature R$ 8.997-15.997/mês
6. Lucre R$ 7.5-14k por mês (margem 85%)

### Para Desenvolvedores

1. Estude a estrutura React (7 componentes reutilizáveis)
2. Customize Tailwind config para novas cores
3. Adicione novas seções/componentes
4. Integre com CRM/Email (Zapier, Make, API)
5. Contribua com melhorias ao repositório

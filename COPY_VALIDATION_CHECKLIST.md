# ✅ Copy Validation Checklist - Cialdini Framework

Validação de copy usando frameworks de persuasão. Cada seção deve passar em 80%+ dos critérios.

---

## 📍 Portfolio-Showcase

### 1. Header/Hero
- [ ] **Problema mencionado explicitamente?** ("Você investe em tráfego pago mas LP não converte?")
- [ ] **Sem jargão técnico?** (Copy em português simples)
- [ ] **Ação clara?** ("Agendar Demonstração" não "Saiba Mais")
- [ ] **CTA em posição visível?** (Top right do header)
- [ ] **Subheadline reforça valor?** (3-7% vs 1-2%)

**Status:** ✅ PASSA (4/5)

---

### 2. Hero Stats Section
- [ ] **Reciprocidade?** ("Você recebe X em troca de Y")
- [ ] **Autoridade?** (3-7% é 3x melhor que mercado)
- [ ] **Prova social?** (500+ consultores/coaches)
- [ ] **Urgência?** ("5 vagas este mês")
- [ ] **Escassez?** (Limitação clara)

**Status:** ✅ PASSA (4/5)

---

### 3. Templates Grid
- [ ] **Cada template tem conversão destacada?** (3-7%, 4-6%, etc)
- [ ] **Cada template fala para nicho específico?** (Não genérico)
- [ ] **CTA é claro?** (Não ambíguo)
- [ ] **Cores diferenciadas?** (Fácil distinção)
- [ ] **"Ideal para" lista é específica?** (Não vaga)

**Status:** ✅ PASSA (5/5)

---

### 4. Pricing Section
- [ ] **Plano "MAIS POPULAR" destacado?** (Efeito âncora)
- [ ] **Preço visualizado claramente?** (Não escondido)
- [ ] **Benefícios diferenciados por plano?** (Não repetidos)
- [ ] **CTA em cada plano?** (Não só em um)
- [ ] **ROI Calculator mostra valor?** (R$ 19.880/mês vs R$ 2.997 investimento)

**Status:** ✅ PASSA (5/5)

---

### 5. Contact Form
- [ ] **Campos obrigatórios são mínimos?** (Só nome + email)
- [ ] **Formulário sinaliza confirmação?** (Pop-up de sucesso)
- [ ] **Chamada clara pós-form?** ("Próximo passo: agendar em Calendly")
- [ ] **Email real visível?** (ewertoncom297@gmail.com)
- [ ] **WhatsApp funcional?** (Link wa.me)

**Status:** ✅ PASSA (5/5)

## 📍 Landing-Consultoria Template

### 1. Hero Section
- [ ] **Headline fala problema?** ("Venda sua consultoria sem gastar R$ 5-8k")
- [ ] **Subheadline promete resultado?** ("Landing page que converte 3-7%")
- [ ] **CTA é ação, não info?** ("Quero minha LP profissional" não "Saiba mais")
- [ ] **Micro-copy reforça?** ("Sem contrato")
- [ ] **Sem buzzwords vazios?** ("Premium", "Elite", "Exclusivo")

**Status:** ✅ PASSA (5/5)

---

### 2. Abertura (Problem Section)
- [ ] **Descreve 4+ dores específicas?** (Custos, agência cara, tempo, conversão)
- [ ] **Cada dor é relatable?** (Não exagerado)
- [ ] **Usa linguagem emocional?** ("Esperando 3-4 semanas" vs "Gastar 3-4 semanas")
- [ ] **Não oferece solução aqui?** (Espera até Benefits)
- [ ] **Financia convicção de que há problema?** (Sim, problema é real)

**Status:** ✅ PASSA (5/5)

---

### 3. Benefits Section
- [ ] **6+ benefícios diferentes?** (Não 3)
- [ ] **Cada benefício tem descrição curta?** (<15 palavras)
- [ ] **Benefícios não mentem?** (Realistas, comprovados)
- [ ] **Fala resultado, não feature?** ("Conversão 3-5x maior" não "Dashboard moderno")
- [ ] **Começa com resultado mais impactante?** (Conversão antes de preço)

**Status:** ✅ PASSA (5/5)

---

### 4. Social Proof
- [ ] **3+ depoimentos com nome/função?** (Não genérico "João")
- [ ] **Cada depoimento tem número?** (4.2%, R$ 3k, etc)
- [ ] **Depoimentos não parecem fake?** (Linguagem natural)
- [ ] **Stats respaldam claims?** (4.8/5 stars real?)
- [ ] **Faces/avatares presentes?** (Aumenta credibilidade)

**Status:** ⚠️ PARCIAL (3/5) - Depoimentos têm placeholders [Cliente 1]

---

### 5. Offer Section
- [ ] **Value stack claro?** (R$ 5.800 "vs" R$ 2.997)
- [ ] **Cada item tem preço?** (R$ 3.500, R$ 1.000, etc)
- [ ] **Desconto é significativo?** (R$ 2.803 = 48% desconto)
- [ ] **Timeline é agressivo?** ("5-7 dias" cria urgência)
- [ ] **Parcelamento oferecido?** ("3x sem juros")

**Status:** ✅ PASSA (5/5)

---

### 6. Objections
- [ ] **3+ objeções principais respondidas?** ("Não tenho tempo", "Já tentei", etc)
- [ ] **Respostas são específicas, não genéricas?** (Não só "Confiamos em você")
- [ ] **Admite verdades concorrentes?** ("LP é coisa de época, agora é community - Verdade")
- [ ] **Cada resposta reforça proposta?** (Sim, mostra diferencial)
- [ ] **Não soa defensivo?** (Tom é educativo)

**Status:** ✅ PASSA (5/5)

---

### 7. Guarantee
- [ ] **Garantia é específica e testável?** ("1 lead qualificado em 15 dias")
- [ ] **Prazo é real?** (15 dias é suficiente)
- [ ] **Risco é transferido para empresa?** (Sim, "devolve 100%")
- [ ] **Sem asteriscos/condições escondidas?** ("Sem perguntas")
- [ ] **Aumenta confiança ou parece risco?** (Aumenta)

**Status:** ✅ PASSA (5/5)

---

### 8. FAQ
- [ ] **6+ perguntas que prospect realmente faz?** (Não obvias)
- [ ] **Respostas são diretas?** (<50 palavras)
- [ ] **Cada resposta vence uma objeção?** (Sim, sistematicamente)
- [ ] **Última pergunta é pré-compra?** ("Posso parcelar?" = quase convertido)
- [ ] **Não responde perguntas que ninguém faz?** (Relevância)

**Status:** ✅ PASSA (5/5)

---

### 9. Urgência/Scarcity
- [ ] **Badge de urgência presente?** ("⏰ 5 vagas disponíveis este mês")
- [ ] **Cria FOMO mas não fake?** (Honesto)
- [ ] **Aparece em 2+ lugares?** (Hero + CTA)
- [ ] **Número é crível?** (5 vagas, não 1.000)
- [ ] **Timeline é real?** ("Depois: fila de espera" = possível)

**Status:** ✅ PASSA (5/5)

---

### 10. P.S.
- [ ] **Reforça benefício principal?** ("Crescem 3-5x mais rápido")
- [ ] **Cria última chance de conversão?** (Sim)
- [ ] **Não é clickbait?** (Relacionado ao copy)
- [ ] **Separa emoção e lógica?** ("Não é talent, é ferramenta")

**Status:** ✅ PASSA (4/5)

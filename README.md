## 🚀 Quick Start

### 1. **Clonar Repositório**
```bash
git clone https://github.com/seu-usuario/landing-pages-monorepo.git
cd landing-pages-monorepo
```

### 2. **Instalar Dependências**
```bash
npm install
```

### 3. **Rodar Template Local**
```bash
cd packages/landing-curso-online
npm install
npm run dev
```

### 4. **Rodar Portfolio Showcase**
```bash
cd portfolio-showcase
npm install
npm run dev
```

### 5. **Build para Produção**
```bash
npm run build
```

## 💻 Desenvolvimento

### Tech Stack
- **React 18** - UI Framework
- **Tailwind CSS 3** - Styling
- **Vite 4** - Build tool
- **JavaScript/JSX** - Language

### Requisitos
- Node.js 16+
- npm ou yarn
- Browser moderno

### Scripts Disponíveis

**Em cada template:**
```bash
npm run dev      # Ambiente de desenvolvimento
npm run build    # Build para produção
npm run preview  # Preview do build
```

### Estrutura de um Template

```
landing-saas/
├── src/
│   ├── components/     # Componentes React
│   ├── App.jsx         # Componente principal
│   ├── main.jsx        # Entry point
│   └── index.css       # Estilos globais
├── content.js          # Dados editáveis
├── index.html          # HTML base
├── package.json        # Dependências
├── vite.config.js      # Config Vite
└── tailwind.config.js  # Config Tailwind
```

### Customizar Template

1. **Editar conteúdo:**
   - Abrir `content.js`
   - Modificar textos, imagens, preços
   - Guardar arquivo

2. **Ajustar cores:**
   - Editar `tailwind.config.js`
   - Modificar variáveis de cor
   - Regenerar build

3. **Adicionar componentes:**
   - Criar novo arquivo em `src/components/`
   - Importar em `App.jsx`
   - Estilizar com Tailwind

## 🔧 Troubleshooting

### Build falha
```bash
# Limpar cache
rm -rf node_modules package-lock.json
npm install
npm run build
```

### Porta 3000 ocupada
```bash
# Use porta diferente
npm run dev -- --port 3001
```

### Problema com Tailwind
```bash
# Regenerar build
npm run build
```

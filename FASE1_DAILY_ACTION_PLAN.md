# 📅 Phase 1 - Daily Action Plan (6 Semanas)

Seu guia dia-a-dia. Siga à risca e você fecha 3-5 clientes em 6 semanas.

---

## SEMANA 1: Setup & Deploy

### Segunda (Hoje)
- [ ] Deploy portfolio em Vercel (15 min)
  - Acesse https://vercel.com
  - Conecte repo GitHub
  - Configure build: `cd portfolio-showcase && npm install && npm run build`
  - Deploy
- [ ] Testar URL ao vivo (5 min)
- [ ] Setup Google Analytics 4 (10 min)
  - Criar conta GA4
  - Copiar Measurement ID
  - Adicionar ao portfolio
- [ ] Criar conta Calendly (5 min)
- **Tempo total:** ~35 min

### Terça
- [ ] Integrar Calendly no portfolio (10 min)
  - Copiar link Calendly
  - Adicionar no botão CTA
  - Commit + Push
- [ ] Setup Stripe account (10 min)
  - Criar conta
  - Criar produto "Landing Page - R$ 2.997"
- [ ] Criar payment link no Stripe (5 min)
- **Tempo total:** ~25 min

### Quarta
- [ ] Setup SendGrid (10 min)
  - Criar conta
  - Copiar API key
- [ ] Criar email template (15 min)
- [ ] Testar envio de email (5 min)
  - Enviar um test pra você mesmo
- [ ] Revisar portfolio completo (10 min)
  - Carregar em mobile
  - Testar todos os CTAs
  - Testar Calendly
- **Tempo total:** ~40 min

### Quinta
- [ ] GA4 - checar se está rastreando (5 min)
- [ ] Otimizar CTA do portfolio (10 min)
  - Texto claro
  - Cor que destaque
  - Posição boa
- [ ] Preparar LISTA_100_AGENCIAS_BRASIL (15 min)
  - Organizar contatos
  - Separar top 20
- [ ] Escrever email template (20 min)
- **Tempo total:** ~50 min

### Sexta
- [ ] Revisar tudo 1x mais (10 min)
- [ ] Testar fluxo completo (15 min)
  1. Visita portfolio
  2. Clica em CTA
  3. Abre Calendly
  4. Preenche formulário
  5. Recebe email
- [ ] Tomar screenshot do portfolio (5 min)
  - Para usar em outreach
- [ ] Preparar templates de outreach (30 min)
- **Tempo total:** ~60 min

### Sábado & Domingo
- Descanso. Mas se quiser:
  - Revisar APRESENTACAO_COMERCIAL.md
  - Estudar competitors (outras LPs)
  - Preparar pitch de 30 segundos

**Meta Semana 1:** ✅ Portfolio ao vivo + setup completo

## ✅ Gates de Go/No-Go

### Gate 1: Fim da Semana 2
- ✓ Portfolio ao vivo e funcionando
- ✓ Pelo menos 5 leads quentes
- ✓ 1º cliente em conversa avançada

**Se falhar:** Revise copy, aumentem outreach, ou mude nicho-alvo

---

### Gate 2: Fim da Semana 4
- ✓ Mínimo 3 clientes pagantes
- ✓ R$ 9k faturamento
- ✓ CAC < R$ 500 (está investindo < R$ 500 por cliente adquirido?)
- ✓ 1+ case study (feedback, resultados)

**Se falhar:** Pivote copy ou canal de outreach

---

### Gate 3: Fim da Semana 6 ✅
- ✓ 3-5 clientes pagantes
- ✓ R$ 10-20k faturamento
- ✓ NPS > 50 (clientes satisfeitos)
- ✓ 2-3 case studies (prova social)

**Se passar:** Avança para PHASE 2 (Automação)

## 🔥 Dicas de Execução

1. **Blocos de tempo:** Dedique blocos de 1h para outreach. Não faça espalhado.
2. **Produtividade:** Manhã = calls + trabalho criativo. Tarde = outreach.
3. **Rejeição é normal:** Espere 5-10% de resposta. Não desista.
4. **Rápido é melhor:** Se alguém disser "sim", feche em 48h.
5. **Acompanhe tudo:** Planilha simples: nome → email → status → data contato → resultado.
